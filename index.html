Details

MEBN stack

CloudStack was built using the MEBN stack. Leveraging Express for the backend API, MongoDB for state, Node.js for logic and integration and finally Backbone.js for frontend.

Customizable

Almost everything is easily customizable, frontend to backend. Frontend was built on top of Backbone.js, keeping logic and templating segregated. Backend leverages multiple drivers for integration and a sane configuration file.
